Ten projekt był pierwszym konkretniejszym, którym zdecydowałem się zająć. Przedstawia biznes plan hipotetycznej kawiarni, który przygotowywałem razem z przyjaciółmi na moje studia.
W nim postawiłem sobie wyzwanie by zrobić jak najlepszą statyczną stronę wykorzystując nowo poznane rzeczy i własne pomysły, pokazując moje możliwości, pierwotnie bez responsywności.
Jestem świadom milionów błędów występujących na stronie. Wszystko jest stopniowo poprawiane wraz z moim postępem. Z czasem mam zamiar urozmaicać stronę, tworzyć bardziej optymalny kod, działać na wszystkich przeglądarkach i urządzeniach.

I've decided to challenge myself by developing static website as good as i can, using all the things that i'll learn and my own ideas, to show my skills. Firstly without responsiveness.
